# Segurança e Rede

## Reparo de Rede
Resolve problemas de conexão executando:
* Reset de pilha TCP/IP e Winsock.
* Limpeza de cache DNS e ARP.
* **Seletor de DNS:** Permite trocar rapidamente para Google (8.8.8.8) ou Cloudflare (1.1.1.1).

## Security Manager (Defender)
* **Hardening:** Ativa proteção contra PUAs (Apps indesejados) e aumenta o nível de proteção na nuvem.
* **Scans:** Permite iniciar varreduras rápidas ou completas diretamente pelo toolkit.

## Password Manager
Permite redefinir senhas de usuários locais ou habilitar a conta oculta de **Administrador** sem precisar usar o CMD.
